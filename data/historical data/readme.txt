changt.txt is hourly precipitation data for changtan precipitation gague (1960-1-1 00:00:00 to 2019-12-31 00:00:00) 
ningx.txt is hourly precipitation data for ningxi precipitation gague (1960-1-1 00:00:00 to 2019-12-31 00:00:00) 
changtstormstatistic.txt is month storm properties for changtan based on the precipitation data of Changt.txt 
changtstormstatisticall.txt is all storm events with storm properties  for Changt
ningxstormstatistic.txt is month storm properties for ningx based on the precipitation data of Changt.txt 
ningxstormstatisticall.txt is all storm events with storm properties for ningx
